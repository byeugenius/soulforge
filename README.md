# SoulForge - AI-Powered Terminal IDE

[![License: AGPL-3.0](https://img.shields.io/badge/License-AGPL%203.0-blue.svg)](https://www.gnu.org/licenses/agpl-3.0)
[![Version](https://img.shields.io/badge/version-3.0.0-brightgreen.svg)](https://github.com/proxysoul/soulforge)

SoulForge is a revolutionary terminal-based Integrated Development Environment (IDE) powered by AI, designed to enhance your coding experience with intelligent assistance, seamless workflows, and unparalleled efficiency. Built by proxySoul, SoulForge leverages cutting-edge AI to provide real-time code insights, automation, and productivity tools directly in your terminal.

## Key Features

- **AI-Driven Code Assistance**: Get real-time suggestions, debugging, and code optimization powered by advanced language models.
- **Terminal-Native**: Lightweight and fast, seamlessly integrates with your existing terminal environment.
- **Intelligent Refactoring**: Automate complex refactoring tasks with precision across your entire codebase.
- **Multi-Agent Architecture**: Parallel AI agents handle tasks like research, coding, and testing simultaneously.
- **Customizable Workflows**: Tailor SoulForge to your needs with plugins and personalized settings.
- **Git Integration**: Streamlined version control with built-in commit, diff, and branch management.

## Installation

SoulForge can be easily installed via npm. Ensure you have Node.js and Bun installed, then run the following command:

```bash
bun install -g @proxysoul/soulforge
```

Once installed, launch SoulForge with:

```bash
soulforge
```

or use the shorthand:

```bash
sf
```

*Note*: SoulForge is published on GitHub Packages. Ensure you have the correct registry configured in your `.npmrc` or authenticate accordingly.
