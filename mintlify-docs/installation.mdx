## Prerequisites

| Dependency | Required | Notes |
|-----------|----------|-------|
| **Bun** >= 1.0 | Only for npm/Bun install method | Not needed for prebuilt binary or Homebrew |
| **Neovim** >= 0.11 | Auto-installed on first launch | SoulForge offers to download it if missing |
| **Nerd Font** | Auto-installed on first launch | SoulForge offers to download one if missing |
| **API key** | At least one LLM provider | See below |

### API key

You need at least one LLM provider key:

| Provider | Env Variable | Models |
|----------|-------------|--------|
| Anthropic | `ANTHROPIC_API_KEY` | Claude Opus 4, Sonnet 4, Haiku 3.5 |
| OpenAI | `OPENAI_API_KEY` | GPT-4o, o3, o4-mini |
| xAI | `XAI_API_KEY` | Grok |
| Google | `GOOGLE_GENERATIVE_AI_API_KEY` | Gemini |
| Ollama | *(none -- runs locally)* | Llama, Mistral, Qwen, etc. |

Add to your shell profile (`~/.zshrc` or `~/.bashrc`):

```bash
export ANTHROPIC_API_KEY=sk-ant-...
```

Or use a single [AI Gateway](https://sdk.vercel.ai/docs/ai-sdk-core/provider-management) key for all providers, or the built-in **Proxy provider** to relay through a local CLIProxyAPI instance (no API key needed).

## Install

<Tabs>
  <Tab title="Prebuilt binary (recommended)">
    Download the latest release from [GitHub Releases](https://github.com/ProxySoul/soulforge/releases/latest), extract, and run the installer:

    ```bash
    tar xzf soulforge-*.tar.gz
    cd soulforge-*/
    ./install.sh
    ```

    Installs to `~/.soulforge/` and adds it to your PATH. Then run `soulforge` (or `sf`).
  </Tab>
  <Tab title="Homebrew">
    ```bash
    brew tap proxysoul/tap
    brew install soulforge
    ```
  </Tab>
  <Tab title="npm / pnpm / Bun">
    ```bash
    # npm
    npm install -g @proxysoul/soulforge

    # pnpm
    pnpm add -g @proxysoul/soulforge

    # bun
    bun install -g @proxysoul/soulforge

    # Or run without installing
    npx @proxysoul/soulforge
    ```
  </Tab>
  <Tab title="Build from source">
    Requires [Bun](https://bun.sh) >= 1.0.

    ```bash
    git clone https://github.com/ProxySoul/soulforge.git
    cd soulforge
    bun install

    # Run in development mode
    bun run dev

    # Or build and link globally
    bun run build
    bun link
    soulforge
    ```
  </Tab>
</Tabs>

## First launch

On first launch, SoulForge checks for Neovim and Nerd Fonts. If either is missing, it offers to download and install them from official GitHub releases. This is optional -- you can install them manually if you prefer.

Run [`/setup`](/reference/commands) anytime to re-check prerequisites.

```bash
soulforge                    # Launch -- select a model with Ctrl+L
soulforge --set-key anthropic sk-ant-...   # Or save a key first
soulforge --list-providers   # Check which providers are ready
```
